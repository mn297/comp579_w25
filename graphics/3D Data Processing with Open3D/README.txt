3D Data Processing with Open3D

To run this Jupyter Notebook, create a new conda environment using the environment.yml file provided. Alternatively, you may manually install the necessary packages in the environment by running the following commands (after activating the environment):
1. `conda install -c anaconda jupyter numpy pandas pillow`
2. `pip install open3d==0.16.0`

Launch Jupyter Notebook from the current folder (cd into folder "3D Data Processing with Open3D" and then launch jupyter notebook).
